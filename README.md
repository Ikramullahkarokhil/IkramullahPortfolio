## üöÄ Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Ikramullahkarokhil/IkramullahPortfolio.git

# Navigate to the project directory
cd IkramullahPortfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
